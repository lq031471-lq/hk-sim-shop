# 香港流量卡国内一手平台 - GitHub Pages 部署指南

## 项目说明

这是一个单文件购物系统，包含：
- **客户端**：商品浏览、购物车、下单、支付
- **商家后台**：商品管理、订单管理、销售统计、利润统计

## 安全警告 ⚠️

**部署前请务必修改默认密码！**

默认密码是 `admin888`，所有人都能猜到。

### 修改密码方法

打开 `index.html`，找到第 2 行：
```javascript
const CONFIG={
  adminPassword:'admin888',  // ← 改成你的强密码
```

把 `admin888` 改成至少 12 位的强密码，包含大小写字母和数字。

## 部署步骤（无需命令行）

### 第一步：注册 GitHub 账号

1. 打开 https://github.com/signup
2. 输入邮箱、设置密码、创建用户名
3. 验证邮箱
4. 完成注册

### 第二步：创建仓库

1. 登录 GitHub 后，点击右上角 **+** 号 → **New repository**
2. 仓库名称填写：`hk-sim-shop`（或其他你喜欢的名字）
3. 选择 **Public**（公开）或 **Private**（私有）
4. 勾选 **Add a README file**
5. 点击 **Create repository**

### 第三步：上传文件

1. 在仓库页面，点击 **Add file** → **Upload files**
2. 点击 **choose your files**
3. 选择桌面上的 `index.html` 文件
4. 在下方填写提交信息：`Add shop system`
5. 点击 **Commit changes**

### 第四步：启用 GitHub Pages

1. 点击仓库顶部的 **Settings** 标签
2. 左侧菜单找到 **Pages**（在左侧最下方）
3. 在 **Source** 部分，选择分支：`main`
4. 文件夹选择：`/(root)`
5. 点击 **Save**
6. 等待 1-2 分钟，页面会显示你的网址：`https://你的用户名.github.io/hk-sim-shop`

### 第五步：访问网站

- **客户端**：直接打开上面的网址
- **商家后台**：在网址后面加上 `#/admin`
  例如：`https://你的用户名.github.io/hk-sim-shop#/admin`

## 安全加固建议

### 1. 修改默认密码（必须）
按照上面的方法修改 `adminPassword`

### 2. 设置为私有仓库（推荐）
在仓库 Settings → General → Visibility → Change to private

### 3. 定期备份数据
商家后台的数据保存在浏览器的 localStorage 中，建议：
- 定期导出订单数据
- 重要操作后刷新页面确认数据已保存

### 4. 不要上传敏感信息
- 不要在代码中硬编码真实支付二维码
- 不要上传身份证、银行卡照片
- 不要在 public 仓库暴露 API 密钥

## 网站功能

### 客户端
- 31 款流量卡商品展示
- 分类筛选、搜索
- 购物车管理
- 订单提交（支持顺丰到付/普快包邮/标快包邮）

- 利润统计（支持日期范围筛选）
- 支付二维码管理（支持本地上传）
- 联系方式管理
- 公告管理
- 订单推送（Server酱）
